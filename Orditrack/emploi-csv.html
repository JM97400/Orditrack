Tutoriel : Comment remplir un fichier CSV pour ajouter des PCs disponibles


Salut ! Ce tutoriel va te montrer comment remplir un fichier CSV pour ajouter des PCs au stock. On va se limiter aux PCs "disponibles", donc tu n’as qu’une seule info à entrer : le numéro de série. C’est facile, suis-moi étape par étape ! Étape 1 : Comprendre le fichier CSV Un fichier CSV, c’est juste une liste simple. Pour ajouter des PCs, on a besoin : Une première ligne qui dit numero_serie (c’est l’en-tête). Ensuite, une ligne par PC avec son numéro de série. Exemple de ce qu’on veut : text Envelopper Copier numero_serie PC-001 PC-002 PC-003 Étape 2 : Choisis ta méthode Tu peux utiliser Excel (ou un tableur) ou le Bloc-notes. Je te montre les deux ! Méthode 1 : Avec Excel Ouvre Excel : Lance Excel sur ton ordinateur. Ajoute l’en-tête : Dans la cellule A1, tape numero_serie (exactement comme ça, sans majuscule sur "serie"). Remplis les numéros : Sous l’en-tête (A2, A3, etc.), écris les numéros de série des PCs. Par exemple : A2 : PC-001 A3 : PC-002 A4 : PC-003 Chaque numéro doit être unique ! Sauvegarde : Clique sur Fichier > Enregistrer sous. Nom : pcs_a_ajouter.csv (ou un nom que tu veux). Type : "CSV UTF-8 (séparé par des virgules) (*.csv)". Clique sur Enregistrer. Méthode 2 : Avec le Bloc-notes Ouvre le Bloc-notes : Cherche "Bloc-notes" dans ton menu Démarrer et ouvre-le. Écris les lignes : Tape ça, en allant à la ligne après chaque entrée : text Envelopper Copier numero_serie PC-001 PC-002 PC-003 Sauvegarde : Clique sur Fichier > Enregistrer sous. Nom : pcs_a_ajouter.csv (n’oublie pas le .csv !). Encodage : Choisis UTF-8 dans la liste déroulante. Clique sur Enregistrer. Étape 3 : Vérifie ton fichier Ouvre ton fichier avec le Bloc-notes pour t’assurer qu’il ressemble à ça : text Envelopper Copier numero_serie PC-001 PC-002 PC-003 Pas d’espaces en trop (ex. PC-001, pas PC-001). Mets un numéro différent pour chaque PC.
